Installation

Karna Portfolio is built using Create Next App. Create Next Aapp is nicely documented to start. You don't need to install or configure tools like Webpack or Babel. They are preconfigured and hidden so that you can focus on the code.

Guide

First of all, make sure you have installed Node and npm/ yarn

Yarn package manager is recommended

  • After downloading zip from ThemeForest, unzip it in your desired location.
  • In uncompressed folder you will find folder named karna-portfolio-vX.X.X..Open this folder in console / terminal. 
        # For Yarn (Highly Recommended)
yarn

# For npm
npm install --force --legacy-peer-deps
  • Now to run development server of Karna, run below command:
        # For Yarn 
yarn start

# For npm
npm run start
  • Visit http://localhost:3000/ to check frontend.

Theme Configuration

1. styles/global.css

Main files to customize the theme is styles/global.css and tailwind.config.js. Remember, this template use Tailwind as main CSS framework.

  --color-text-inverted: 255, 255, 255;
  --color-text-base: 255, 255, 255;
  --color-text-button-base: 255, 255, 255;
  --color-text-accent: 46, 196, 182;
  --color-text-button-inverted: 6, 196, 182;

2. tailwind.config.js

Basically this is tailwind configuration file. You can see the official guide here. 
        
module.exports = {
  ...
  theme: {
    extend: {
      textColor: {
        skin: {
          base: withOpacity("--color-text-base"),
          inverted: withOpacity("--color-text-inverted"),
          accent: withOpacity("--color-text-accent"),
          "button-base": withOpacity("--color-text-button-base"),
          "button-inverted": withOpacity("--color-text-button-inverted")
        }
      },
      backgroundColor: {
        skin: {
          base: withOpacity("--color-bg-base"),
          accent: withOpacity("--color-bg-accent"),
          gray: withOpacity("--color-bg-gray"),
          "button-bg-base": withOpacity("--color-button-bg-base"),
          "button-bg-inverted": withOpacity("--color-button-bg-inverted")
        }
      },
      ...
    }
  }
}

We use withOpacity function to create opacity functionality. 

        
const withOpacity = (variableName) => {
  return ({ opacityValue }) => {
    if (opacityValue !== undefined) {
      return `rgba(var(${variableName}), ${opacityValue})`
    }

    return `rgb(var(${variableName}))`
  }
}

We pass variable name we created in global.css file.
